# nari-management（タスク管理アプリ）

単発の作業と、決まった手順（フロー）のある作業を、両方まとめて管理するための自分用アプリです。

## 使い方（かんたん）

### 1. 開く
`index.html` を**ダブルクリック**すると、いつものブラウザ（Safari / Chrome）で開きます。
インターネットは不要。ネットがなくても動きます。

### 2. スマホで使う＆PCと同期する
スマホでアプリとして使い、PCとデータを共有するには **[SETUP.md](SETUP.md)** の手順に沿って設定します（Supabase＋GitHub Pages、どちらも無料）。設定後は、同じメールでログインするだけでPC↔スマホが自動同期されます。

## できること

- **単発タスク**：チェックひとつで完了（例：テーマリサーチ）
- **フロータスク**：決まった手順を順番に進める（例：YouTube＝香盤表作成→撮影→動画選定→編集→編集確認→サムネ作成→投稿設定）
  - タスク作成時に**「撮影日（基準日）」を入れるだけ**で、各工程の期日が自動で入ります
- **今日のおすすめ**：優先度・期日・放置日数から、今日やるべきタスクを提示
- **リマインド**：期限切れ・○日間動いていないタスクを教えてくれる
- **プロジェクト進捗**：各プロジェクトのタスク件数と進捗率を表示。**今週／今月／全期間／カスタム期間**で絞り込めます（先の日付のタスクを登録しても、今の進捗が薄まりません）
- **カラーパレット**：設定タブから4種（リヴィエラ／シトラス／タンジェロ／エーゲ）を選べます
- **テンプレート編集**：設定タブから、工程名や「＋○日」を自由に変更できます

## データの保存場所

- **同期の設定をする前**：データは**この端末のブラウザの中**だけに保存されます（別端末とは共有されません）。
- **同期の設定をした後（[SETUP.md](SETUP.md)）**：クラウド（Supabase）にも保存され、同じメールでログインした端末どうしで自動同期されます。

### バックアップのすすめ
設定タブ →「エクスポート」で、データをファイル（.json）として保存できます。復元は「インポート」から。同期を使う場合も、念のためときどきエクスポートしておくと安心です。
